Belajar Dasar HTML dan CSS

HTML (HyperText Markup Language) adalah bahasa untuk membuat struktur halaman web. CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan dan gaya elemen HTML.

Apa itu HTML?

HTML adalah bahasa markup yang menggunakan elemen seperti:

  • <h1>: Untuk membuat judul.
  • <p>: Untuk membuat paragraf.
  • <a>: Untuk membuat tautan.
  • <img>: Untuk menambahkan gambar.

Contoh Kode HTML:

<!DOCTYPE html>
<html>
  <head>
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
    <p>Ini adalah paragraf pertama saya.</p>
    <a href="https://google.com">Kunjungi Google</a>
  </body>
</html>

Apa itu CSS?

CSS digunakan untuk mempercantik tampilan halaman web. Misalnya, Anda dapat mengubah warna teks, mengatur tata letak, atau menambahkan animasi.

Contoh Kode CSS:

body {
  background-color: #f0f8ff;
  color: #333;
  font-family: Arial, sans-serif;
}

h1 {
  color: #0077cc;
  text-align: center;
}

Struktur File HTML dan CSS

Biasanya, HTML dan CSS dipisahkan menjadi dua file:

File HTML:

<!DOCTYPE html>
<html>
  <head>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <h1>Halo Dunia!</h1>
  </body>
</html>

File CSS:

body {
  background-color: #eef;
  color: #333;
}

h1 {
  color: #0077cc;
}
